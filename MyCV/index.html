Pokemon Collection Django App (Hyperion Catsptone Poject)

Overview

This application is a Django-based platform designed to serve as a comprehensive database for Pokémon Trading Card collections. Utilizing the Pokémon TCG API, the app allows users to navigate through various Pokémon card sets that are stored in a SQL database. Users can create accounts, log in, and log out to personalize their experience, adding or removing cards from their collection as they wish.

Features

  • Browse Pokémon Card Sets: Navigate through a variety of Pokémon trading card collections (sets) retrieved via the Pokémon TCG API.
  • Detailed Card View: Within each set, view detailed information and images for each individual card.
  • Add to Collection: Logged-in users can add cards from any set to their personal collection.
  • Remove from Collection: Cards can also be easily removed from a user's collection if they are logged in.
  • User Authentication: Create an account, log in, and log out to personalize and save your card collection.

Twitch Chrome Extension (Front end and Back end)

Overview

This is a google chrome extension that I made using twitch API. This extension is designed to provide an enhanced experience for Twitch users, allowing quick access to information about streamers, their current status, and their social media links. It is geared to bring real-time updates right to your browser's toolbar.

Features

  • Icon Color Notification: The extension icon turns colorful when the streamer is live, making it easy to know the stream status at a glance.
  • Live Stream Thumbnail: A thumbnail featuring a screenshot of the ongoing live stream appears when the streamer is live, which is updated every 5 minutes.
  • Direct Stream Access: The live stream thumbnail is clickable, providing direct access to the current live stream.
  • Social Media Links: Easily access all of the streamer's social media platforms via the extension.
  • Offline Support: When the streamer is not live, the icon turns black and white, and clicking on the thumbnail redirects to the streamer's Twitch channel.
